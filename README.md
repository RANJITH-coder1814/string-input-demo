# string-input-demo
A simple C++ program demonstrating how to take string input from the user using cin and getline(). It highlights the difference between cin (which reads input until a space) and getline() (which reads the full line including spaces).
# String Input Demo

A simple C++ program demonstrating **string input** from the user using `cin` and `getline()`.  

## Features
- Accepts a string from the user
- Demonstrates the difference between `cin` and `getline()`
- Prints the entered string to the console
